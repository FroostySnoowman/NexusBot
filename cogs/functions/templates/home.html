Endpoint 1

This endpoint allows you to retrieve information about a user. Here is an example request:

GET /users/123 HTTP/1.1
Host: api.nexusservices.xyz/users/123
Login: api.nexusservices.xyz/login

And here is an example response:

HTTP/1.1 200 OK
Content-Type: application/json
 
{
    "description": "Software Developer",
    "portfolio": "https://nexusservices.xyz/",
    "timezone": "MST",
    "paypal": "example@example.com",
    "paypalme": "https://paypal.me/example"
}

Host: example.com
Content-Type: application/json
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxMjN9.ztK5z5WYl5mOogRyK0tH8Usm0k4-M4cAGSD1nI69p8o
 
{
	"username": "janedoe",
	"email": "janedoe@example.com",
	"password": "p@ssw0rd"
}

And here is an example response:

HTTP/1.1 201 Created
Content-Type: application/json
 
{
	"id": 124,
	"username": "janedoe",
	"email": "janedoe@example.com"
}
